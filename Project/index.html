Sintaxi bàsica: declaració de Variables i Operadors

En aquest apartat veurem un petit resum de com es declaren les variables i la seva utilitat. Així com la usualitat i comprensió del operadors per a jquery.

Declaració d'una variable
La seva sintaxi es: var nom_variable = 'cadena de caracters/valor/funció'.
varnom1= 'hola a tots' ;


Declaració de més d'una variable
La seva sintaxi es la mateixa. Unicament les separament amb una ","
varnom1= 'hola a tots' ,
varvalor= 5 ,
varfuncio1= function(){console.log("hola")} ;
la tabulació millora la lectura del codi, però no té cap significat especial per el jquery

Operadors bàsics
Els operadors bàsics permeten manipular el valors. Tenim diferets tipus : +, -, * , /,... que combinats entre ells podem obtenir diferents lectures.

Concatenació
varfoo= 'Hola' ;
varbar= 'Tots' ;
console.log(foo + ' ' + bar);// la consola de depuració mostra 'Hola Tots'
Multiplicació i divisió
varfoo= 2 * 3 ;
varbar= 4 / 2 ;
console.log(foo + ' ' + bar);// la consola de depuració mostra '8'
Augmentar i decrementar abans de una operació
var i = 1 ;
var j = ++i ;// incrementació previa: j es igual a 2; i es igual a 2
var k = i++ ;// incrementació posterios: k es igual a 2; i es igual a 3
Obligar a una cadena de caràcters actui com a un nombre.
varfoo= 1 ;
varbar= '2' ;
El constructor 'Number' obliga a la cadena a actua com a numero
console.log(foo + Number(bar));// la consola de depuració mostra '3'
Operadors lògics (AND i OR)
Els operadors lògics permeten d'avaluar una sèrie d'operands. Donant com a solució verdader (true) o fals (false).L'operador OR (||) retorna el valor de l'operand primer, si això és cert; en cas contrari torna l'operand segon. Si ambdós operands són falses retorna fals (false). L'operardor AND (&&) retorna el valor de l'operand primer si això és fals; cas contrari torna l'operand segon. Quan ambdós valors són veritables retorna cert (true), però retorna fals. Veiem diferents casos.
var foo = 1 ;
var bar = 0 ;
var baz = 2 ;
foo||bar;// retorna 1, el cual es verdader (true)
bar||foo;// retorna 1, el cual es verdader (true)
foo&&bar;// retorna 0, el cual es falso (false)
foo&&baz;// retorna 2, el cual es verdader (true)
baz&&foo;// retorna 1, el cual es verdader (true)
Operadors de comparació
Els operadors de comparació permeten comparar si certs valors són idèntics o equivalents. var foo = 1 ;
var bar = 0 ;
foo==bar;// el significa 'igual'
foo!=bar;// el significa 'diferent'
foo===bar;// el significa 'identic'
foo===bar;// el significa 'no identic'
foo&bar;// el significa 'major que'
foo&=bar;// el significa 'major o igual que'
foo&bar;// el significa 'menor que'
foo&=bar;// el significa 'menor o igual que'
